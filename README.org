** GitHubでのコンフリクトの解消

#+begin_src bash
# ローカルのmasterを最新に
git checkout master
git pull
# shizukaに移動してマージ
git checkout shizuka
git merge master
# コンフリクトを解消する
git add .
git commit -m 'Merge'
git push -u origin shizuka
#+end_src
